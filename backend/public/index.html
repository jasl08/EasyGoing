What is EasyGoing?

EasyGoing is a smart trip-planning web app that helps users build and organize travel itineraries with the help of AI and real-time Google Maps data. Whether you're a solo traveler or planning a group adventure, EasyGoing makes it easy to find destinations, track budgets, and explore new places efficiently.

Key Features

  • Optimized trip itinerary to minimize distance and time traveled
  • AI-powered travel suggestions using OpenAI
  • Smart autocomplete, directions, and place search with Google Maps
  • An interactive map interface capable of displaying optimized routes across multiple destinations with support for various modes of transportation simultaneously
  • Integrated budgeting system to track trip costs
  • Photo caching with Amazon S3 for fast, low-latency image loading

Technologies Used

Node.js · Express · Firebase Authentication · PostgreSQL · Google Maps API · OpenAI API · Amazon S3 · React Native (mobile)
